Морской бой — проект на JavaScript, где создаётся браузерная версия игры для двух игроков на сетке 7x7. Игроки размещают корабли и по очереди стреляют по координатам, угадывая позиции противника.

Основные компоненты
Две сетки: "Морское поле" (скрытые корабли) и "Целевое поле" (метки выстрелов).

Корабли: 3 корабля по 3 клетки (всего 9 клеток на поле).
